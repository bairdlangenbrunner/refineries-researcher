# data/

- `main_*.parquet` — the built main database (timestamped). Regenerated by `merge.py`;
  gitignored by default (decide per-repo whether to commit a snapshot).
- Raw source downloads (RMI xlsx, OGIM gpkg, etc.) live here or under `sources/<name>/data/`
  and are **gitignored** — the manifest is the committed recipe, not the bytes.

Download raw sources here before ingesting; see each `sources/<name>/manifest.yml` for the
Drive id / URL / sibling path.
